# Utoo Landing Page

Landing page for [Utoo](https://github.com/utooland/utoo) - An Unified Toolchain For  Web Development.

## Features

- Showcase of three core packages: `utoo`, `@utoo/pack`, `@utoo/web`
- Interactive terminal demos
- Browser-based playground powered by `@utoo/web` (WASM)
- i18n support (English / Chinese)
- Dark / Light theme

## Tech Stack

- [Next.js 15](https://nextjs.org/) with App Router
- [Tailwind CSS](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
- [Monaco Editor](https://microsoft.github.io/monaco-editor/)
- [@utoo/web](https://github.com/utooland/utoo/tree/next/packages/utoo-web) for browser builds

## Development

```bash
# Install dependencies
ut install

# Start dev server
ut dev

# Build for production
ut build
```
