# utoo CLI

`utoo` (or `ut`) is a fast, modern package manager for JavaScript projects.

## Commands

### install

Install all dependencies from `package.json`:

```bash
ut install
# or
ut i
```

Install specific packages:

```bash
ut install lodash
ut install -D typescript  # dev dependency
```

### run

Run scripts defined in `package.json`:

```bash
ut run build
ut run dev
# or simply
ut build
ut dev
```

### Other Commands

| Command | Alias | Description |
|---------|-------|-------------|
| `ut list` | `ut ls` | List installed packages |
| `ut view <pkg>` | `ut v` | View package info |
| `ut update` | `ut u` | Update dependencies |
| `ut deps` | `ut d` | Generate lock file |
