# Introduction

Welcome to the **Utoo** documentation!

Utoo is a unified frontend toolchain that provides:

- **utoo** - A fast, modern package manager
- **@utoo/pack** - A high-performance bundler
- **@utoo/web** - Browser-native builds with zero config

## Quick Start

```bash
# Install utoo globally
npm install -g utoo

# Install dependencies in your project
ut install

# Run development server
ut dev
```

## Features

- Lightning fast dependency resolution
- Native ESM support
- Zero-config bundling
- Browser-native module loading
