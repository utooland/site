# Getting Started

This guide will help you get up and running with Utoo.

## Installation

Install utoo globally using npm:

```bash
npm install -g utoo
```

Or use it directly with npx:

```bash
npx utoo install
```

## Creating a New Project

```bash
# Create a new directory
mkdir my-app && cd my-app

# Initialize package.json
npm init -y

# Install dependencies with utoo
ut install react react-dom
```

## Project Structure

A typical Utoo project looks like this:

```
my-app/
├── src/
│   ├── index.tsx
│   └── App.tsx
├── package.json
└── utoopack.json
```
