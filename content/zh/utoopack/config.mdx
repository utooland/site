# 配置

import { Callout, Tabs } from 'nextra/components'

`@utoo/pack` 可通过 `utoopack.json`、`utoopack.config.mjs` 或编程 API 配置。

## 配置文件

<Tabs items={['utoopack.json', 'utoopack.config.mjs']}>
  <Tabs.Tab>
```json filename="utoopack.json"
{
  "$schema": "@utoo/pack/config_schema.json",
  "mode": "production",
  "entry": [
    {
      "import": "./src/index.ts",
      "html": {
        "template": "./index.html"
      }
    }
  ],
  "output": {
    "path": "./dist",
    "filename": "[name].[contenthash:8].js",
    "chunkFilename": "[name].[contenthash:8].js",
    "clean": true
  },
  "sourceMaps": true
}
```
  </Tabs.Tab>
  <Tabs.Tab>
```javascript filename="utoopack.config.mjs"
import { defineConfig } from '@utoo/pack';

export default defineConfig({
  mode: "production",
  entry: [
    {
      import: "./src/index.ts",
      html: { template: "./index.html" }
    }
  ],
  output: {
    path: "./dist",
    filename: "[name].[contenthash:8].js",
    clean: true
  },
  sourceMaps: true
});
```

## 核心选项

### mode

构建模式，影响优化和默认值。

| 值 | 描述 |
|----|------|
| `development` | 更快构建，更好调试，启用 HMR |
| `production` | 优化输出，压缩，tree shaking |

### entry

定义应用入口。

```json
{
  "entry": [
    {
      "name": "main",
      "import": "./src/index.ts",
      "html": {
        "template": "./index.html",
        "filename": "index.html",
        "title": "My App",
        "inject": "body",
        "scriptLoading": "defer"
      },
      "library": {
        "name": "MyLib",
        "export": ["default"]
      }
    }
  ]
}
```

### output

配置输出设置。

```json
{
  "output": {
    "path": "./dist",
    "filename": "[name].[contenthash:8].js",
    "chunkFilename": "[name].[contenthash:8].js",
    "publicPath": "/",
    "clean": true,
    "copy": ["public"],
    "type": "standalone",
    "chunkLoadingGlobal": "TURBOPACK",
    "entryRootExport": "MyApp"
  }
}
```

| 选项 | 描述 |
|------|------|
| `path` | 输出目录 |
| `filename` | 入口 chunk 文件名模板 |
| `chunkFilename` | 非入口 chunk 文件名模板 |
| `cssFilename` | CSS 文件名模板 |
| `cssChunkFilename` | 非入口 CSS chunk 文件名模板 |
| `assetModuleFilename` | 资源模块文件名模板 |
| `publicPath` | 资源的公共 URL 路径 |
| `clean` | 构建前清理输出目录 |
| `copy` | 复制静态文件（字符串路径或 `{ from, to }` 对象） |
| `type` | 输出类型：`standalone` 或 `export` |
| `chunkLoadingGlobal` | 加载 chunk 的全局变量名 |
| `entryRootExport` | 将入口导出暴露到全局作用域 |

### target

目标环境字符串（如 `"web"`、`"node"`）。

### define

构建时变量替换。

```json
{
  "define": {
    "process.env.NODE_ENV": "\"production\"",
    "__VERSION__": "\"1.0.0\""
  }
}
```

### externals

从打包中排除依赖。

```json
{
  "externals": {
    "react": "React",
    "react-dom": "ReactDOM"
  }
}
```

支持简单字符串全局变量、UMD 配置和高级子路径配置。

### resolve

模块解析配置。

```json
{
  "resolve": {
    "alias": {
      "@": "./src",
      "@components": "./src/components"
    },
    "extensions": [".ts", ".tsx", ".js", ".jsx"]
  }
}
```

### module

自定义 loader 的模块规则和条件。

```json
{
  "module": {
    "rules": {
      "*.svg": "svgr-loader",
      "*.mdx": [
        { "loader": "mdx-loader", "options": {} }
      ]
    },
    "conditions": {
      "*.js": {
        "path": { "type": "glob", "value": "**/node_modules/**" }
      }
    }
  }
}
```

| 选项 | 描述 |
|------|------|
| `rules` | 将 glob 模式映射到 loader |
| `conditions` | 按路径（glob 或 regex）配置 loader 条件 |

### provider

自动提供模块（类似 webpack ProvidePlugin）。

```json
{
  "provider": {
    "$": "jquery",
    "Buffer": ["buffer", "Buffer"]
  }
}
```

## 优化

```json
{
  "optimization": {
    "minify": true,
    "treeShaking": true,
    "concatenateModules": true,
    "moduleIds": "deterministic",
    "removeConsole": true,
    "removeUnusedExports": true,
    "removeUnusedImports": true,
    "splitChunks": {
      "js": {
        "minChunkSize": 50000,
        "maxChunkCountPerGroup": 40,
        "maxMergeChunkSize": 200000
      }
    },
    "modularizeImports": {
      "lodash": {
        "transform": "lodash/{{member}}",
        "preventFullImport": true
      }
    },
    "packageImports": ["antd", "lodash-es"],
    "transpilePackages": ["my-local-package"],
    "nestedAsyncChunking": true,
    "wasmAsAsset": false
  }
}
```

| 选项 | 描述 |
|------|------|
| `minify` | 启用代码压缩 |
| `treeShaking` | 移除未使用代码 |
| `concatenateModules` | 模块合并（scope hoisting） |
| `moduleIds` | ID 策略：`named` 或 `deterministic` |
| `noMangling` | 禁用名称混淆 |
| `removeConsole` | 移除 console 语句（`true` 或 `{ exclude: ["error"] }`） |
| `removeUnusedExports` | 移除未使用的导出 |
| `removeUnusedImports` | 移除未使用的导入 |
| `splitChunks` | Chunk 分割配置 |
| `modularizeImports` | 转换 barrel 导入（类似 `babel-plugin-import`） |
| `packageImports` | 要优化导入的包 |
| `transpilePackages` | 转译指定包 |
| `nestedAsyncChunking` | 启用嵌套异步 chunking |
| `wasmAsAsset` | 将 WASM 打包为资源 |

## 样式

```json
{
  "styles": {
    "autoCssModules": true,
    "emotion": true,
    "styledComponents": true,
    "styledJsx": true,
    "less": { "implementation": "less" },
    "sass": { "implementation": "sass" },
    "inlineCss": {
      "injectType": "styleTag"
    }
  }
}
```

| 选项 | 描述 |
|------|------|
| `autoCssModules` | 自动检测并应用 CSS Modules |
| `emotion` | 启用 @emotion/react 转换 |
| `styledComponents` | 启用 styled-components 转换（`true` 或配置对象） |
| `styledJsx` | 启用 styled-jsx 转换（`true` 或 `{ useLightningcss: true }`） |
| `less` | Less 预处理器配置（`implementation` 及 Less 选项） |
| `sass` | Sass/SCSS 预处理器配置（`implementation` 及 Sass 选项） |
| `inlineCss` | 通过 style 标签注入 CSS（`insert`、`injectType`） |

## 图片

```json
{
  "images": {
    "inlineLimit": 10000
  }
}
```

| 选项 | 描述 |
|------|------|
| `inlineLimit` | 小于此大小（字节）的图片内联为 base64 |

## 开发服务器

```json
{
  "devServer": {
    "hot": true,
    "port": 3000,
    "host": "localhost",
    "https": false
  }
}
```

| 选项 | 描述 |
|------|------|
| `hot` | 启用热模块替换 |
| `port` | 监听端口 |
| `host` | 绑定主机 |
| `https` | 使用 HTTPS |

## React

```json
{
  "react": {
    "runtime": "automatic",
    "importSource": "react"
  }
}
```

| 选项 | 描述 |
|------|------|
| `runtime` | JSX 运行时：`automatic` 或 `classic` |
| `importSource` | JSX 导入来源（默认：`react`） |

## defineEnv

按目标环境注入环境变量（用于 Next.js 集成）。

```json
{
  "defineEnv": {
    "client": [
      { "name": "API_URL", "value": "https://api.example.com" }
    ],
    "nodejs": [
      { "name": "DB_HOST", "value": "localhost" }
    ],
    "edge": []
  }
}
```

| 目标 | 描述 |
|------|------|
| `client` | 浏览器构建中可用的变量 |
| `nodejs` | Node.js 服务端构建中可用的变量 |
| `edge` | Edge 运行时构建中可用的变量 |

## 实验性功能

实验性功能（可能在未通知的情况下更改）。

```json
{
  "experimental": {
    "swcPlugins": [["@swc/plugin-emotion", {}]],
    "reactCompiler": true,
    "esmExternals": true,
    "viewTransition": true
  }
}
```

| 选项 | 类型 | 描述 |
|------|------|------|
| `swcPlugins` | `[string, any][]` | SWC 转换插件 |
| `reactCompiler` | `boolean \| object` | 启用 React Compiler |
| `esmExternals` | `boolean \| "loose"` | ESM externals 处理 |
| `dynamicIO` | `boolean` | Dynamic IO 支持 |
| `useCache` | `boolean` | 启用缓存 |
| `cacheHandlers` | `Record<string, string>` | 自定义缓存处理器 |
| `ppr` | `boolean \| "incremental"` | 部分预渲染 |
| `taint` | `boolean` | Taint 追踪 |
| `viewTransition` | `boolean` | View Transitions API |
| `serverActions` | `boolean \| object` | Server Actions 支持 |
| `mdxRs` | `object` | MDX-RS 选项 |

## 高级选项

| 选项 | 类型 | 描述 |
|------|------|------|
| `sourceMaps` | `boolean` | 启用 source maps |
| `stats` | `boolean` | 启用构建统计输出 |
| `nodePolyfill` | `boolean` | 为浏览器 polyfill Node.js 内置模块 |
| `persistentCaching` | `boolean` | 启用持久化构建缓存 |
| `cacheHandler` | `string` | 自定义缓存处理器模块路径 |
| `pluginRuntimeStrategy` | `string` | 插件执行方式：`workerThreads` 或 `childProcesses` |

## 运行时选项

这些选项可在 `utoopack.config.mjs` 中或通过编程 API 设置：

| 选项 | 描述 |
|------|------|
| `processEnv` | 编译时环境变量 |
| `watch` | 文件监听（`{ enable, pollIntervalMs }`） |
| `dev` | 是否以开发模式运行 |
| `buildId` | 构建标识符 |
| `packPath` | `@utoo/pack` 的绝对路径 |
| `rootPath` | 文件嵌套的根路径 |
| `projectPath` | 根路径内的项目路径 |
