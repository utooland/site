# pm-pack

从包目录创建一个 tarball 压缩包。

**别名:** `pk`

## 用法

```bash filename="Terminal"
ut pm-pack [OPTIONS] [PATH]
```

## 参数

| 参数 | 描述 |
|------|------|
| `[PATH]` | 包目录路径（默认为当前目录） |

## 示例

```bash filename="Terminal" copy
# 打包当前目录
ut pm-pack
ut pk

# 打包指定目录
ut pm-pack ./packages/my-lib

# 预览打包内容（不生成文件）
ut pm-pack --dry-run
```

## 选项

| 选项 | 描述 |
|------|------|
| `--dry-run` | 预览模式，不实际生成 tarball |

## 使用场景

- 发布前检查包的内容和大小
- 手动创建 `.tgz` 包用于本地测试
- 配合 `--dry-run` 确认哪些文件会被包含
