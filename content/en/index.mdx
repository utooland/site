# Overview

Welcome to the **Utoo** documentation!

Utoo is a unified frontend toolchain combining three integrated components:

| Package | Description |
|---------|-------------|
| [utoo](/en/docs/utoo) | Fast, npm-compatible package manager |
| [@utoo/pack](/en/docs/utoopack) | High-performance bundler with HMR |
| [@utoo/web](/en/docs/utooweb) | Browser-native bundling via WebAssembly |

## Quick Start

```bash filename="Terminal" copy
# Install utoo globally
npm install -g utoo

# Install project dependencies
ut install

# Run scripts
ut dev
ut build

# Execute remote packages
utx create-react-app my-app
```

## Why Utoo?

- **Fast** - Rust-powered performance
- **Compatible** - Drop-in npm replacement
- **Unified** - One toolchain for all needs
- **Modern** - Native ESM, TypeScript, and more
