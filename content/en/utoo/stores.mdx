## Paths Overview

| Path | Purpose |
|------|---------|
| `~/.utoo/config.toml` | Global configuration file |
| `~/.utoo/cache/` | Downloaded package tarballs |
| `~/.utoo/global/` | Global package installations |

## Package Cache

When you install a package, utoo caches the tarball in `~/.utoo/cache/`:

<Callout>
  Cached packages are reused across projects, speeding up subsequent installations.
</Callout>

### Clearing Cache

<Tabs items={['Clear All', 'Clear Specific', 'Check Location']}>
  <Tabs.Tab>
```bash filename="Terminal" copy
ut clean
```
  </Tabs.Tab>
  <Tabs.Tab>
```bash filename="Terminal" copy
ut clean lodash
ut clean react*
```
  </Tabs.Tab>
  <Tabs.Tab>
```bash filename="Terminal" copy
ut config get cache-dir
```
  </Tabs.Tab>
</Tabs>

## Global Packages

Globally installed packages are stored in `~/.utoo/global/`:

```bash filename="Terminal" copy
# Install globally
ut install -g typescript

# Location
# ~/.utoo/global/node_modules/typescript/
# ~/.utoo/global/node_modules/.bin/tsc
```

## Project Files

In each project, utoo uses:

<FileTree>
  <FileTree.Folder name="my-project" defaultOpen>

| File | Purpose |
|------|---------|
| `package.json` | Dependencies manifest |
| `package-lock.json` | Dependency lock file |
| `node_modules/` | Installed packages |
| `.utoo.toml` | Project-specific config (optional) |
