# logout

Logout from npm registry.

**Alias:** `lo`

## Usage

```bash filename="Terminal"
ut logout [OPTIONS]
```

## Examples

```bash filename="Terminal" copy
# Logout from default registry
ut logout
ut lo

# Logout from a specific registry
ut logout --registry https://registry.npmjs.org
```

## Options

| Option | Description |
|--------|-------------|
| `--registry <url>` | Specify the registry to logout from |

## How It Works

1. Reads the saved authentication token
2. Sends a logout request to revoke the token
3. Removes the token from local storage
