# list

List installed dependencies.

**Alias:** `ls`

## Usage

```bash filename="Terminal"
ut list <PACKAGE>
```

## Arguments

| Argument | Description |
|----------|-------------|
| `<PACKAGE>` | Package name to show dependencies for (required) |

## Examples

```bash filename="Terminal" copy
# List specific package and its dependencies
ut list react
ut ls lodash
```
