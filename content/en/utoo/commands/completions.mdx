# completions

Generate shell completion scripts for utoo.

**Alias:** `cmp`

## Usage

```bash filename="Terminal"
ut completions [SHELL]
```

## Arguments

| Argument | Description |
|----------|-------------|
| `[SHELL]` | Shell to generate completions for (auto-detected if omitted) |

Supported shells: `bash`, `zsh`, `fish`, `powershell`, `elvish`

## Examples

```bash filename="Terminal" copy
# Auto-detect shell and output completions
ut completions

# Generate for a specific shell
ut completions bash
ut completions zsh
ut completions fish
```

## Setup

Add to your shell config for persistent completions:

```bash filename="~/.bashrc" copy
eval "$(utoo completions bash)"
```

```bash filename="~/.zshrc" copy
eval "$(utoo completions zsh)"
```

```bash filename="~/.config/fish/completions/utoo.fish" copy
utoo completions fish > ~/.config/fish/completions/utoo.fish
```
